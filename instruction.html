Правильная регистрация аккаунта для нашего приложения ✅

1. Перейти на сайт - Жми

2. При регистрации вводим промокод - GPT24. Это важно для подключения аккаунта.

3. Активируем аккаунт двумя депозитами по 1001 и 1000 рублей.

4. Играем в Lucky Jet, чтобы программа могла начать выдавать сигналы.

5. Используем ваш ID в программе для запуска.
